# Portfolio Website

A modern, responsive portfolio website built with [Next.js](https://nextjs.org), showcasing professional projects, skills, and services.

## Features

- **Responsive Design**: Fully responsive layout optimized for all devices
- **Project Showcase**: Display professional projects with details and descriptions
- **Skills Section**: Highlight technical expertise and tools
- **Services Section**: Outline professional services offered
- **Contact Form**: Integrated contact form for inquiries
- **Dark Mode Support**: Theme switching capability
- **SEO Optimized**: Built-in SEO optimization with sitemap and robots.txt

## Tech Stack

- **Framework**: Next.js with TypeScript
- **Styling**: Tailwind CSS with PostCSS
- **Form Handling**: Server actions for contact forms
- **Linting**: ESLint for code quality

## Getting Started

### Prerequisites

- Node.js 20 installed
- npm or yarn package manager

### Installation & Development

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
npm run start
```

## Project Structure

- `/app` - Next.js app router and pages
- `/components` - Reusable React components
- `/content` - JSON data for projects and testimonials
- `/actions` - Server actions for forms
- `/assets` - Icons and images
- `/public` - Static files and documents
- `/hooks` - Custom React hooks
- `/utils` - Helper functions and utilities
- `/lib` - Type definitions
